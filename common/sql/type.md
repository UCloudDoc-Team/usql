# 数据类型

USQL支持以下数据类型：

| 数据类型   | 描述     | 范围                  |                  
| INT    | 整型     | -2 ^ 31 ～ 2 ^ 31-1 |
| STRING | 字符串    | -                  |                  
| DOUBLE | 双精度浮点型 | 在NULL情况下，采用计算值默认值为0 |                  
| BIGINT | 有符号整型  | -2 ^ 63 ～ 2 ^ 63-1 |                  
| DATE   | 日期类型   | YYYY-MM-DD          |
